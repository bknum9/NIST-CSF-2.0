<h1>Project Overview</h1>

<p>This project is a full NIST Cybersecurity Framework (CSF) 2.0 policy implementation for a fictional small business scenario — Golden Ridge Veterinary Group (GRVG), a regional veterinary practice operating four clinic locations across central Virginia. The project was completed as a hands-on educational exercise to simulate real-world cybersecurity program development for a small organization with no existing formal security posture. The engagement was triggered by a business email compromise incident and culminated in a complete, structured cybersecurity documentation package built from the ground up. The project began with a Current Profile Assessment conducted in February 2026, which benchmarked GRVG at Tier 1 across all six CSF 2.0 Functions, establishing the baseline for all policy work.</p>

<p>The deliverables include 23 policy and procedure documents mapped directly to NIST CSF 2.0 subcategories, a master Information Security Policy, a standalone Incident Management Plan, four network architecture diagrams covering each clinic location, and a ControlMap-supported compliance assessment tracking GRVG's implementation status against each control. Every policy document addresses its corresponding CSF outcomes (e.g., GV.OC, PR.AA, RS.AN). It is written at a level appropriate for a small business with no dedicated IT staff — practical, specific to GRVG's tools and environment (Microsoft 365, eVetPractice, consumer-grade networking), and designed to be immediately actionable. Taken together, the project demonstrates the end-to-end process of assessing, documenting, and implementing a cybersecurity program aligned to a recognized industry framework.</p>
